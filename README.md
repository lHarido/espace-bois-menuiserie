# ESPACE BOIS MENUISERIE

Site vitrine Next.js pour ESPACE BOIS MENUISERIE, menuiserie bois, PVC et aluminium à Confort.

## Installation

```bash
npm install
```

## Lancement en local

```bash
npm run dev
```

Le site sera disponible sur `http://localhost:3000`.

## Build de production

```bash
npm run build
```

## Variables à prévoir

Créer un fichier `.env.local` si besoin :

```bash
CONTACT_EMAIL=adresse-a-renseigner@example.com
NEXT_PUBLIC_SITE_URL=https://domaine-a-renseigner.fr
```

## À compléter avant mise en ligne

- Adresse e-mail réelle de réception du formulaire.
- Hébergeur réel dans les mentions légales.
- Photos réelles de réalisations, avec autorisation.
- Logo définitif si l'entreprise en possède un.
- URL Facebook exacte uniquement si elle est confirmée.
- Vérification de la certification Qualibat RGE.
- Vérification des informations SIREN/SIRET, avis et notes sur les sources officielles.
- Connexion du formulaire à un service réel : API route, Formspree, Resend, Nodemailer ou autre.

## Sources à vérifier

- Annuaire des entreprises : https://annuaire-entreprises.data.gouv.fr/entreprise/801054768
- INSEE / Sirene : https://avis-situation-sirene.insee.fr/
- PagesJaunes : https://www.pagesjaunes.fr/
- Qualibat : https://www.qualibat.com/
